title=[B41] Role Display System
description=A lightweight, server-friendly way to show player roles across chat, nameplates, and maps. Designed for [b]multiplayer servers[/b] with [b]admin hierarchies[/b] or [b]custom communities[/b].
description=
description=[h1]What it does[/h1]
description=[list]
description=[*] [b]Chat[/b]: Adds colored role tags before usernames in messages.
description=[*] [b]Nameplates[/b]: Renders role tags next to player usernames above their heads with distance-based fading.
description=[*] [b]Map + Mini-Map[/b]: Draws role tags near player markers with a subtle background for readability; uses vehicle position when driving.
description=[*] [b]Multi-role support[/b]: Show all roles or only the highest priority.
description=[*] [b]Formatting[/b]: Choose bracket styles: [Role], (Role), {Role}, <Role>, Role:, or none. Per-role color via RGB.
description=[/list]
description=
description=[h1]Admin tools[/h1]
description=Open [b]Admin Panel > Role Manager[/b] to configure:
description=[list]
description=[*] Create / rename / delete roles.
description=[*] Set color, bracket style, and numeric priority.
description=[*] Assign players via [b]Online list[/b] or [b]Manual input[/b] (usernames separated by semicolons).
description=[*] See player counts per role and update in real time when changes are pushed.
description=[/list]
description=
description=[h1]Visibility & rules[/h1]
description=[list]
description=[*] Respects [b]DisplayUserName[/b], [b]ShowFirstAndLastName[/b], and [b]MouseOverToSeeDisplayName[/b] server settings.
description=[*] Enforces [b]line of sight[/b] for world rendering and hides [b]invisible[/b] players from non-admins.
description=[*] Map rendering follows [b]MapRemotePlayerVisibility[/b] server settings.
description=[/list]
description=
description=[h1]Sandbox options[/h1]
description=[list]
description=[*] [b]Chat Integration[/b]: enable/disable chat role tags.
description=[*] [b]Map Integration[/b]: enable/disable map role tags.
description=[*] [b]Show Multiple Roles[/b]: show all roles or only the highest priority.
description=[/list]
description=[i]Note: World nameplate role display is always on and follows server visibility rules.[/i]
description=
description=[h1]Performance[/h1]
description=[list]
description=[*] Client-side caching for chat and rendering.
description=[*] Distance-based fading and off-screen cleanup.
description=[/list]
description=
description=[h1]How to use[/h1]
description=[list]
description=[*] In-game as admin: open [b]Admin Panel[/b] and click [b]Role Manager[/b].
description=[*] Create one or more roles; set color, brackets, and priority.
description=[*] Add players (online list or manual usernames) to each role.
description=[*] Optionally adjust Sandbox: Chat/Map integrations and Multi-role display.
description=[/list]
description=
description=[h1]Known notes[/h1]
description=[list]
description=[*] Role changes may take a few seconds to propagate due to client caching.
description=[*] Very long role names can affect positioning in rare cases.
description=[/list]
description=
